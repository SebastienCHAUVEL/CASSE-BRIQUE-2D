## Description

Réalisation d'un jeu de casse brique en pur javascript, rendu avec la balise HTML `<canvas>`.

Pour réaliser ce projet, j'ai suivi le [tutoriel de MDN](https://developer.mozilla.org/fr/docs/Games/Tutorials/2D_Breakout_game_pure_JavaScript)
